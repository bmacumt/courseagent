# 前后端联调方案 v1.0

> 文档日期：2026-04-18
> 前端：`courseagent/frontend/` (React + TypeScript + Vite)
> 后端：`courseagent/backend/` (FastAPI + SQLite + ChromaDB)

---

## 一、联调环境

### 启动命令

```bash
# 终端1: 后端
cd courseagent/backend
source .venv/bin/activate
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 终端2: 前端
cd courseagent/frontend
pnpm dev
```

## 五、已知限制

1. **报告导航**: 前端使用 submission_id 作为 report_id 进行导航（`/teacher/reports/${sub.id}`）。当 submission_id 与 report_id 不一致时，报告页面会显示"加载失败"
2. **QA 无文档**: ChromaDB 为空时，QA 返回"未找到参考资料"，属正常行为
3. **评分依赖 LLM API**: 需要有效的 LLM_API_KEY 才能正常评分
