## 1. 设计理念

**关键词：** 学术感、低饱和度、信息密度适中、操作直觉化

本系统面向高校隧道工程课程的教学辅助场景，UI 设计需体现：
- **专业克制** — 低饱和度配色，避免花哨装饰，让内容本身成为焦点
- **操作直觉** — 符合用户对教育系统的心理模型：左侧导航 + 右侧内容区
- **状态透明** — 评分进度、提交状态等异步操作需有清晰的视觉反馈
- **响应式优先** — 学生可能使用手机查看作业和报告，教师/管理员以桌面端为主

### 2.3 通信层设计

```
前端 (Vue)  ←── Axios + JWT Bearer ──→  后端 (FastAPI)
     │                                      │
     │  请求拦截器：                          │
     │  - 注入 Authorization header           │
     │  - Loading 状态                        │
     │                                        │
     │  响应拦截器：                          │
     │  - 401 → 跳转登录页                    │
     │  - 403 → 提示无权限                    │
     │  - 500 → 全局错误提示                   │
```

Axios 实例：

```typescript
// src/utils/request.ts
const api = axios.create({
  baseURL: import.meta.env.VITE_API_BASE || 'http://localhost:8000',
  timeout: 30000,
})

// 请求拦截：注入 token
api.interceptors.request.use(config => {
  const auth = useAuthStore()
  if (auth.token) {
    config.headers.Authorization = `Bearer ${auth.token}`
  }
  return config
})

// 响应拦截：统一错误处理
api.interceptors.response.use(
  res => res.data,
  err => {
    if (err.response?.status === 401) {
      const auth = useAuthStore()
      auth.logout()
      router.push('/login')
    }
    ElMessage.error(err.response?.data?.detail || '请求失败')
    return Promise.reject(err)
  }
)
```

### 3.4 组件风格

- **卡片**：白色背景 + 1px #E8ECF0 边框 + 8px 圆角，无阴影（或仅 0 1px 3px rgba(0,0,0,0.04)）

- **表格**：斑马纹（#FAFBFC 隔行变色），无边框，底部分割线
- **标签/Tag**：圆角 4px，低饱和底色 + 深色文字
- **输入框**：1px #DCDFE6 边框，聚焦时 Primary 色边框

## 6. 全局交互规范

### 6.1 状态反馈

| 场景 | 组件 | 样式 |
|------|------|------|
| 操作成功 | ElMessage.success | 绿色提示，3 秒自动消失 |
| 操作失败 | ElMessage.error | 红色提示，不自动消失 |
| 确认操作 | ElMessageBox.confirm | 标题 + 描述 + 确认/取消 |
| 表单验证 | 字段下方红色文字 | 实时校验，提交前检查 |
| 加载中 | ElLoading / v-loading | 区域遮罩 + 旋转图标 |
| 评分中 | 状态标签 + 轮询 | 蓝色标签 + "AI 评分中" 文字 |

### 6.2 Token 管理

```
登录成功 → localStorage 存储 token + user info
每次请求 → Axios 拦截器注入 Authorization header
Token 过期 → 401 响应 → 清除 localStorage → 跳转 /login
页面刷新 → 检查 localStorage 有无 token → 有则验证 GET /me → 失败则跳转登录
```

### 6.3 路由守卫

```typescript
router.beforeEach((to, from, next) => {
  const auth = useAuthStore()

  // 白名单路由（无需登录）
  if (to.path === '/login') {
    next()
    return
  }

  // 未登录 → 跳转登录
  if (!auth.token) {
    next('/login')
    return
  }

  // 角色校验
  const requiredRole = to.meta.role as string | undefined
  if (requiredRole && auth.role !== requiredRole && auth.role !== 'admin') {
    ElMessage.error('无权限访问')
    next(from.path)
    return
  }

  next()
})
```
